# Student Expense Tracker

## Overview
A simple full-stack web application to track daily expenses.

## Features
- Add expenses
- View expense list
- Delete expenses

## Technologies Used
- HTML, CSS, JavaScript
- Node.js, Express
- MongoDB

## How to Run
1. Install dependencies using `npm install`
2. Start MongoDB
3. Run server using `node server.js`
4. Open browser at `http://localhost:3000`
